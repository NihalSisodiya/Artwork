Artworks DataTable (React + PrimeReact) **https://artworksapidatatable.netlify.app/**

This project is built as part of a React. It demonstrates how to integrate the Art Institute of Chicago API with a modern PrimeReact DataTable to display artwork information with server-side pagination and persistent row selection.

Features
- Vite + TypeScript setup for fast development and type safety
- PrimeReact DataTable for a modern, responsive UI
- Server-side pagination: fetches data per page from the API (https://api.artic.edu/api/v1/artworks)
- Row selection with checkboxes: select/deselect individual rows or all rows on the current page
- Persistent selection: selections remain when navigating between pages
- Custom selection overlay: allows selecting a custom number of rows from the current page
- Modern UI styling: striped rows, hover effects, and gridlines


Tech Stack
- React (with TypeScript)
- Vite for fast bundling
- PrimeReact 10.x for UI components
- PrimeIcons for icons
- Axios for API requests
